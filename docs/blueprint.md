# **App Name**: HeartConnect

## Core Features:

- One-on-One Calls: Enable users to make 1-on-1 calls with others.
- Private Chat: Enable users to send text messages to one another in private chat.
- Conversation Tags: Allow users to add tags like 'Feeling Nervous' or 'Breakup Relief' to signal the type of conversation they are seeking.
- Instant Call Feature: Display a list of available users and filter by voice tone or vibe.
- Coin-Based Talk Time: Use a coin-based system where users spend coins for each minute of talk time, with options to recharge via UPI, Paytm, or promo offers.
- Anonymous Mode: Let users hide their name or use a nickname for expressing themselves freely.
- Voice Intro Evaluation: Employ an AI tool to evaluate submitted voice intro profiles for positive or negative sentiment and adherence to guidelines.

## Style Guidelines:

- Primary color: Soft pastel pink (#FFB6C1) for a friendly and inviting feel.
- Background color: Very light pastel rose (#FFE4E1), almost white.
- Accent color: Light pastel blue (#ADD8E6) for highlighting interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a blend of modernity and warmth, suitable for both headings and body text.
- Utilize a set of simple, rounded icons that contribute to the friendly feel, designed to complement the colors in the pallette.
- Employ a soft, rounded design aesthetic with a focus on approachability, achieved through rounded buttons and conversational UI elements like chat bubbles.
- Subtle, gentle animations on button presses and loading states, giving a polished user experience.
